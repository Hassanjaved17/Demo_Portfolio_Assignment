## ✨ About The Project

**Picto Portfolio** is a sleek, modern, and fully responsive personal portfolio website designed to showcase my work, skills, and frontend development journey.

It focuses on **clean UI, smooth animations, and performance optimization** — exactly what recruiters look for.

Deployed using **GitHub Pages**

1. Go to Repository Settings
2. Click on "Pages"
3. Select `main` branch
4. Your site goes live instantly 🚀
